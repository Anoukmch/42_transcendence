# Transcendence

This template should help get you started developing with Vue 3 in Vite.

## Recommended IDE Setup

[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (and disable Vetur)

## Environment
You can specify the enviromnet inside the ```.env``` file.

### Development Environment

If ```FRONTEND_ENV=dev``` then a development vite container with hot reloading is used.

### Production Environmnet

If ```FRONTEND_ENV=prod``` then a nginx container will serve the bundled source files. (No hot reloading).
